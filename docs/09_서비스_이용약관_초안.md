# 서비스 이용약관 초안

최종 갱신: 2026-07-21 KST

이 문서는 `어디고` 배포 전 검토용 초안입니다. 실제 배포 전 사업자 정보와 공개 URL을 확정해야 합니다.

## 1. 목적

이 약관은 어디고 서비스의 이용 조건, 사용자와 서비스 제공자의 권리와 의무, 서비스 이용 제한, 데이터 처리 기준을 정합니다.

## 2. 서비스 내용

어디고는 다음 기능을 제공합니다.

- 여행 취향 선택
- 여행 성향 분석
- 국내 여행지 추천
- 관광지 기본 정보 확인
- 지도 앱 연결

AI 추천 결과와 외부 관광정보는 참고용입니다. 방문 가능 여부, 운영 시간, 요금, 교통 상황은 사용자가 방문 전 직접 확인해야 합니다.

## 3. 계정

- 서비스는 Apps in Toss 또는 토스 로그인 기반 식별 정보를 사용할 수 있습니다.
- 사용자는 본인 계정을 안전하게 관리해야 합니다.
- 서비스는 보안상 필요한 경우 이용 제한 또는 재인증을 요구할 수 있습니다.

## 4. 사용자 의무

사용자는 다음 행위를 해서는 안 됩니다.

- 타인의 정보를 무단 사용
- 허위 정보 입력
- 서비스 장애를 유발하는 반복 요청
- 크롤링, 역공학, 비정상 API 호출
- 서비스를 부정한 목적 또는 영리 목적으로 무단 이용

## 5. 외부 API와 AI

서비스는 한국관광공사 관광정보 API와 AI 서비스를 활용할 수 있습니다. 외부 API 장애, 정보 변경, AI 응답 오류로 인해 추천 결과가 부정확할 수 있습니다.

## 6. 서비스 변경과 중단

서비스는 기능 개선, 보안, 장애 대응, 외부 API 변경, 비용 문제로 일부 기능을 변경하거나 중단할 수 있습니다. 가능한 경우 앱 또는 공지 채널을 통해 안내합니다.

## 7. 탈퇴와 삭제

사용자는 회원탈퇴를 요청할 수 있습니다. 탈퇴 시 계정 정보와 이용 기록은 삭제 또는 비식별 처리됩니다. 법령상 보관이 필요한 정보는 해당 기간 동안 보관할 수 있습니다.

## 8. 면책

추천 결과는 여행지 선택을 돕기 위한 참고 정보입니다. 실제 여행 결정, 이동, 예약, 비용 지출, 안전 확인은 사용자의 책임으로 이루어집니다.
