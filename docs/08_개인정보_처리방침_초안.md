# 개인정보 처리방침 초안

최종 갱신: 2026-07-12 KST

이 문서는 `어디고` 배포 전 검토용 초안입니다. 실제 배포 전 사업자명, 대표자, 주소, 공개 URL을 확정해야 합니다.

## 1. 처리 목적

어디고는 다음 목적을 위해 개인정보 또는 이용 정보를 처리할 수 있습니다.

- 사용자 식별 및 서비스 이용 관리
- 여행 취향 분석
- 국내 여행지 추천
- 한국관광공사 관광정보 조회
- AI 기반 추천 문구 생성
- 오류 분석, 보안 점검, 서비스 품질 개선

## 2. 처리 항목

필수 또는 자동 처리 가능 항목:

- 앱인토스 익명 사용자 키를 서버에서 재해시한 값과 추천 세션 식별자
- 서비스 이용 기록
- 접속 로그, 기기 및 브라우저 정보
- 사용자가 선택한 7개 여행 취향 답변과 태그
- 관광지 후보 수, 최종 추천 결과, 이미지·지도 제공 여부, 추천 처리 시간

선택 항목:

- 사용자가 직접 선택한 지역
- 위치 기반 추천 허용 시 위치 정보

## 3. 개인정보 최소화

- 이름, 연락처, 주민등록번호 등 직접 식별 정보는 추천에 사용하지 않습니다.
- AI에는 추천에 필요한 취향 답변과 관광지 후보 정보만 전달합니다.
- 위치 정보는 사용자가 허용하거나 직접 선택한 범위에서만 사용합니다.
- 정확한 위치 좌표와 앱인토스 익명 사용자 키 원문은 추천 품질 로그에 저장하지 않습니다.

## 4. 외부 처리

서비스 제공을 위해 아래 외부 서비스를 사용할 수 있습니다.

- 한국관광공사 국문 관광정보 서비스_GW API: 관광지 정보 조회
- Google Gemini 등 AI 서비스: 여행 성향 분석과 추천 문구 생성
- Vercel 등 배포/호스팅 서비스: 약관 페이지 또는 서버 기능 제공

## 5. 보관 기간

- 회원 정보: 회원 탈퇴 또는 이용 목적 달성 시 삭제 또는 비식별 처리
- 추천 품질 로그: 재해시된 익명 키, 여행 취향 답변, 후보와 추천 결과를 생성일부터 90일 보관 후 삭제
- 오류/분석 로그: 서비스 안정화와 보안 점검에 필요한 기간 보관 후 삭제
- 법령상 보관이 필요한 정보는 해당 기간 동안 보관
