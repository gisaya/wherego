   - `crowd` 1개는 필수 포함
   - `mobility` 또는 `accessibility` 중 1개는 필수 포함

## 운영 기준

질문은 "재미"보다 "검색에 쓰이는 태그"가 우선이다. 각 선택지는 최소 하나 이상의 구조화 태그를 가져야 한다.

예:

```json
{
  "label": "숲에서 쉬고 싶음",
  "tags": ["forest", "healing", "nature", "quiet"],
  "searchHints": {
    "keywords": ["수목원", "숲", "자연휴양림", "생태공원"]
  }
}
```

Gemini에는 원문 답변만 넘기지 말고 `tags`, `searchHints`, `constraints`를 함께 넘긴다. 그래야 Gemini가 매번 다른 말로 흔들리지 않고 한국관광공사 API function calling을 안정적으로 수행한다.
