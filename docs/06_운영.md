```text

```

```text

```

빌드 명령:

```powershell

```

## Git

현재 remote:

```text
origin https://github.com/gisaya/wherego.git
```

사용자가 "저장"이라고 하면 `SAVE_PROTOCOL.md`에 따라 문서 갱신, 검증, commit, push까지 진행한다.

## API 환경변수

```text
PUBLIC_DATA_PORTAL_SERVICE_KEY
KTO_KOR_SERVICE_DOC_URL
KTO_KOR_SERVICE_ENDPOINT
KTO_DATALAB_SERVICE_DOC_URL
KTO_DATALAB_SERVICE_ENDPOINT
WHEREGO_USAGE_LIMIT_ENABLED=true
```

원문 인증키는 `.env.local`에만 두고 Git에 커밋하지 않는다.

앱 추천 횟수 정책은 기본 3회/일, 리워드 광고 최대 10회/일, 친구 공유 3회/일 1회다. 공유 기능을 활성화하려면 앱인토스 콘솔에서 공유 리워드 승인을 받고 `src/config.ts`의 `WHEREGO_SHARE_REWARD_MODULE_ID`에 발급된 모듈 ID를 입력한다.
