# 프로젝트 현황

최종 갱신: 2026-07-21 KST

## 제품

`어디고`는 사용자가 7개의 선택형 질문에 답하면 한국관광공사 관광정보와 지역별 방문자수 데이터를 바탕으로 AI가 국내 관광지 1곳을 추천하는 Apps in Toss 미니앱이다.

핵심 가치는 성격 테스트가 아니라 실제로 갈 목적지를 빠르게 좁히는 것이다.

## 사용자 흐름

1. 일반 `/` 또는 혜택 `/promotion` 진입 후 횟수·로그인·결제·상품 초기 조회
2. 현재 위치나 직접 선택 지역으로 출발 기준 설정
3. 서버가 만든 질문 7개에 카드 방식으로 답변
4. 서버가 KorService2/DataLab 후보 준비
5. 결과 전 전면광고 노출
6. Gemini가 후보 5~7개에서 최종 1개 선택. 후보가 0개면 장소를 직접 제안하고 관광공사에서 재검증
7. 결과 카드 저장 또는 네이버지도 열기

광고가 닫혔는데 AI 응답이 남아 있으면 전용 분석 화면을 표시한다. 일시 API 실패만 같은 세션으로 재시도하고, 검증 후보가 없으면 출발지를 유지한 채 새 질문을 고른다.

## 질문 정책

- 원천 3개: 이동 범위, 동행/제약, 목적지 의도
- 핵심 일반 3개: 서로 다른 검색 영향 테마
- 재미 일반 1개: 약한 가중치지만 실제 관광지 메타데이터에 연결
- 한 세트 안에서 질문 ID와 의미 테마가 중복되지 않는다.
- 운영 질문은 `crowd:mobility:accessibility = 6:2:2` 가중치로 고르며 `crowd`를 우선한다.
- 질문 카피와 메타데이터는 서버 리소스만 수정한다.

세부 정책은 `10_선택지_질문풀_검토.md`를 따른다.

## 추천과 데이터

- 관광정보: 한국관광공사 국문 관광정보 서비스_GW `KorService2`
- 지역 방문 신호: 한국관광공사 빅데이터 지역별 방문자수_GW `DataLabService`
- 최종 선택: `gemini-3.1-flash-lite`
- 지도: 네이버지도 검색 링크
- 이동 시간: 직선거리 기반 추정치이며 길찾기 API 결과가 아니다.
- 최소 거리 하한이 있으면 검색 범위를 전국으로 넓힌 뒤 거리 필터를 적용한다.
- 방문자수는 지역 단위 참고 신호이며 실시간 장소 혼잡도로 표현하지 않는다.

## 횟수와 수익화

- 기본 AI 추천: KST 하루 2회
- 리워드 광고: +1회, 하루 최대 2회
- 친구 공유: +3회, 하루 1회
- 인앱결제: 소모성 AI 추천 10회권, 미사용 횟수 만료 없음
- 일일 크레딧을 먼저 사용하고 유료 크레딧을 나중에 사용한다.
- 무료·광고·공유 크레딧에는 결과 전 전면광고를 노출하고 유료 크레딧에는 생략한다.

## 프로모션

- 일반 진입 `intoss://wherego`는 프로모션을 실행하지 않는다.
- 혜택 진입 `intoss://wherego/promotion`에서만 결과 프로모션을 실행한다.
- 운영 혜택은 토스 포인트 50원, 1인 1회다.
- 로컬 Storage, 세션 중복 방지, 서버 원자적 예약을 함께 사용한다.
- 서버 예약 장애 시 SDK를 호출하지 않는 fail-closed 정책이다.

## 앱 화면

- TDS Provider, Text, Button을 기본으로 사용한다.
- 선택 카드와 지역 카드는 고정 크기 `Pressable`로 구현한다.
- 2지선다는 2열 카드, 4지선다는 2x2 카드다.
- 질문 배너는 선택지 수와 무관하게 하단 안전영역 위에 고정하고 카드 영역이 가리지 않도록 여백을 둔다. AI/결과 화면은 기존 하단 배너 영역을 사용한다.
- 첫 화면은 횟수·저장 로그인·결제 복원·이용권 상품 조회가 끝난 뒤 한 번에 표시한다.
- 성공 결과 2회 이상 사용자에게 한 번만 앱 리뷰 요청 CTA를 표시한다.
- 저장 카드는 1080x1350 PNG이며 공유 API를 호출하지 않는다.
- 모든 화면의 뒤로가기는 종료 확인창을 거친다.

## 출시 전 남은 확인

1. 일반/혜택 진입을 각각 QR 테스트한다.
2. 프로모션 1회 지급과 중복 방지 실패 시 미지급을 확인한다.
3. 광고·공유·구매 횟수와 로그인 복원을 실기기에서 확인한다.
4. PNG 저장, 이미지, 지도, 고정 배너, 리뷰 CTA, 뒤로가기를 최신 Toss 앱에서 회귀한다.
5. 3시간 QC와 보조 24시간 표본으로 추천 실패, 이미지 누락과 목적지 집중도를 관찰한다.
